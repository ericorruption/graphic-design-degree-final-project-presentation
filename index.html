O que sabemos

Motivação

Pergunta projetual

Como melhorar a manutenção e expansão do site institucional UniRitter, considerando conceitos de ubiquidade e sua utilização em multiplataforma?

Objetivo geral

Projetar guia de estilo baseado na biblioteca de componentes Bootstrap, para ser utilizado no portal da instituição de ensino superior UniRitter, considerando os conceitos de responsividade para melhorar a experiência do usuário em múltiplas plataformas, ainda melhorando o processo de manutenção do site.

Objetivos específicos

  • Produzir guia de estilos para casos de expansão do site com novas secções e conteúdos
  • Investigar design responsivo, guia de estilos front-end e bibliotecas de componentes de interface
  • Projetar e desenvolver exemplos de aplicações da biblioteca de componentes para o futuro site responsivo do UniRitter

  if( supports3DTransforms ) {

    var nodes = document.querySelectorAll( selector );

    for( var i = 0, len = nodes.length; i < len; i++ ) {
      var node = nodes[i];

      if( !node.className ) {
        node.className += ' roll';

    }
  }
